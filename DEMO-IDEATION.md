## 1. 단편 데모 (Simple / Wow)

각 항목: **훅 → 조작 → 보이는 변화 → 플래그 → 콘텐츠 앵글**

### S1. 복리의 폭주 (Compound interest)
- **훅:** “한 달 커피값, 30년 뒤?”
- **조작:** 원금·이율·복리 주기 슬라이더
- **변화:** 곡선이 꺾이며 최종액 `<Stat>`이 뛰어오름
- **플래그:** 2D
- **앵글:** 금융 문해력 / “공식 암기 → 만져보기”
- **런칭:** ✅ 1순위 (가장 싸고 확실)

### S2. 케플러 쓸기 (Equal areas)
- **훅:** “근일점에서 왜 빨라지나 — 면적이 말한다”
- **조작:** 이심률 드래그, 궤도 위 행성 스크럽
- **변화:** 같은 시간 부채꼴 면적이 항상 같음 (`mood="space"`)
- **플래그:** `--3d`
- **앵글:** 슬라이드 vs 3D Before/After
- **런칭:** ✅ 1순위 (3D mood 증명)

### S3. 갈톤 보드 500알
- **훅:** “정규분포를 코딩하지 않았다 — 떨어뜨렸을 뿐”
- **조작:** 떨어뜨리기 / 리셋, (선택) 못 개수
- **변화:** 알갱이가 쌓여 종 모양
- **플래그:** `--physics`
- **앵글:** “물리 엔진 = 설명”
- **런칭:** ✅ 강추 (클립 바이럴성 최고)

### S4. 이진 탐색 + 튜터
- **훅:** “튜터한테 답 물어보면 힌트만 준다”
- **조작:** 배열에서 탐색 구간 좁히기 + 챗
- **변화:** 구간 하이라이트 + 소크라테스 응답 (퀴즈 답 비누출)
- **플래그:** `--tutor`
- **앵글:** 그라운딩/환각 방지 = 세일즈 피처
- **런칭:** ✅ 필수 (tutor 레이어 증명) · Vercel 호스트

### S5. 파도 간섭 (2-source)
- **훅:** “두 돌멩이, 보강·상쇄가 눈에 보임”
- **조작:** 파장·거리·위상 슬라이더
- **변화:** 간섭 무늬 실시간
- **플래그:** 2D (또는 `--3d` 수면)
- **앵글:** 수능 파동 / 공학 입문
- **런칭:** 예비 (수능 커리큘럼과 연결 시 승격)

### S6. Rule of 72 / 반감기
- **훅:** “72÷이율 = 두 배 기간”을 타이머로
- **조작:** 이율 또는 감쇠 상수
- **변화:** 타임라인 위 마커가 움직임
- **플래그:** 2D
- **앵글:** S1의 동생 콘텐츠 (시리즈화)
- **런칭:** 콘텐츠 시리즈용, 데모 슬롯 필수는 아님

### S7. DNA 전사 스크럽
- **훅:** “코돈이 단백질로 읽히는 테이프”
- **조작:** `<Scrubber>`로 염기 진행
- **변화:** 코돈→아미노산 매핑이 한 글자씩
- **플래그:** 2D (`mood` 불필요) / 셀 비주얼이면 `--3d` `cell`
- **앵글:** 생물 선생님 타겟
- **런칭:** 세그먼트 확장용

**단편 런칭 픽:** **S1 · S2 · S3 · S4** (금융 / 우주3D / 물리엔진 / 튜터)

## 3. 메인 후보 A — 「수능 물리: 시험장의 물리」

### 로그라인
기출의 **그림·조건**을 만지는 랩으로 바꾸고, 단원 맵을 깨며 오답 패턴을 줄인다.

### 왜 Faraday인가
수능 물리는 원래 “상황 그림 + 변수”인데 PDF로는 고정됨.  
슬라이더·스크럽·3D 궤도가 **문항의 실험 장치**가 됨. 과외 대학생 페르소나(GTM Stage1)와 직결.

### 월드 톤
- 팩: `map2dPack` (시험장/단원 복도) → 여력 되면 `world3dPack` “플라네타륨 복도”
- 비주얼: 문제지 여백·흑백 도식에서 → 조작 시 컬러로 “채점되는” 느낌 (과한 게임 UI 지양)
- 각 노드 클리어 = 기출형 `<Quiz>` + `useNode().complete()`

### 노드 맵 (6–7개, 선수 관계)

```
[0 측정·단위] ──┬── [1 등가속도] ── [2 포물선] ──┐
                │                                ├── [5 역학 에너지] ── [6 모의 미니셋]
                └── [3 뉴턴·힘] ── [4 원운동] ──┘
```

| 노드 | 레슨 한 줄 | 조작 | 플래그 |
|---|---|---|---|
| 0 측정 | 유효숫자·단위 환산이 점수를 먹는다 | 자리수/단위 토글 | 2D |
| 1 등가속 | v-t 아래 넓이 = 변위 | 그래프 스크럽 | 2D |
| 2 포물선 | 각도만 바꿔 사거리 최대 | θ 슬라이더 + 궤적 | 2D/`physics` |
| 3 뉴턴 | 알짜힘 방향과 가속 | 힘 벡터 합치기 | 2D |
| 4 원운 | 줄에 매달린 추 — 장력 vs 중심력 | ω·r | 2D/`physics` |
| 5 에너지 | 마찰 있으면 역학적 에너지 어디로 | 높이·μ | 2D |
| 6 미니셋 | 섞은 3문항 (튜터 힌트만) | Quiz + `--tutor` | tutor |

### 콘텐츠 훅
- “수능 물리 그 그림, 손가락으로 움직이면?”
- “오답노트 대신 랩 클리어”
- Before/After: 실제 기출 캡처(공정 이용·출처) vs Faraday 랩

### 리스크
- 문항 텍스트 무단 복제 금지 → **숫자·상황만 오마주**, “기출 스타일” 자작
- 국내 채널(에브리타임·수능 커뮤니티) 톤에 맞춤

## 4. 메인 후보 B — 「항해 일지: 상대성·중력·시간」 (인터스텔라 모티프)

### 로그라인
블랙홀 근처에서 **시간이 다르게 흐른다**는 감각을, 노드를 항해하며 실험으로 이해한다.

### 왜 Faraday인가
영화가 이미 대중 언어를 만들어 둠. Faraday는 “CG 감상”을 “랩”으로 바꿈.  
`mood="space"` + 3D 커리큘럼이 제품 샷에 최적.

### 월드 톤
- 팩: `world3dPack` (성좌) 또는 `map2dPack` “항해 차트”
- 노드 = 행성/스테이션. 클리어 시 항로 해금
- 카피·UI에 영화 고유명·대사·점수 사용 금지. 물리 개념명만.

### 노드 맵 (6개)

```
[1 케플러 궤도] ── [2 중력 슬링샷] ── [3 등가원리 엘리베이터]
         │                                    │
         └──── [4 중력 시간지연] ── [5 빛 휨·렌즈] ── [6 최종: 신호 동기화]
```

| 노드 | 레슨 | 조작 | 플래그 |
|---|---|---|---|
| 1 궤도 | 케플러 2법칙 복습 | 이심률 | `--3d` |
| 2 슬링샷 | 접근 속도·각도 → 탈출 | 벡터 | `--3d`/`physics` |
| 3 등가 | 가속 엘리베이터 = 중력 | 가속 토글 | 2D |
| 4 시간지연 | 깊은 우물일수록 시계가 느림 | 중력 포텐셜 슬라이더 | 2D+차트 |
| 5 렌즈 | 광원·질량 위치 → 아인슈타인 링 느낌 | 질량·시선 | `--3d` |
| 6 동기화 | 두 시계 맞추기 미션 (퀴즈/튜터) | Quiz + tutor | `--tutor` |

### 콘텐츠 훅
- “그 우주 영화 보고 난 뒤, 손으로 돌려보는 상대성”
- 클립: 노드 맵 줌아웃 → 행성 클릭 → 시간지연 슬라이더

### 리스크
- IP: 비공식 팬 랩 톤, 상표 회피
- 상대론 오개념 위험 → 각 노드 `<Callout>` + 튜터 grounding 텍스트 필수
