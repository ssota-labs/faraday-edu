# Pack `lecture-design` — quality bar

Design-quality rules for lessons/curricula built with this pack. These grade the
*teaching*, not the code.

- **Aligned.** Every lesson and every check maps to an outcome verb; orphan
  content (maps to no outcome) is cut. (backward-design)
- **Gated on mastery.** Progression is gated on demonstrated mastery, not
  exposure; prerequisites are real. (prerequisite-graph)
- **Generative.** The core interaction has the learner predict/construct/explain,
  not just watch or scrub; novices get worked-examples-and-fade. (generative-learning)
- **Retrieval built in.** Units end in a recall check that needs the interaction
  to answer; later nodes re-quiz earlier ideas. (spaced-retrieval)
- **Feed-forward.** Hints say where-to-next, not just right/wrong. (feedback)
- **One methodology.** A single audience-appropriate method leads the lesson; no
  mechanical stacking of all five moves on every node, no learning-styles variants.
