# sim2d vs game2d

| Need | Pack |
|------|------|
| x(t) from equations, graphs, vectors | **sim2d** (SVG + GSAP) |
| Collisions, sprites, tilemaps, game feel | **game2d** (Pixi + Matter) |
| 3D orbits, molecules | **three** |
| Course map navigation | **map2d** |

Never use Matter to "simulate" constant acceleration when `x = v₀t + ½at²` is the learning goal.
