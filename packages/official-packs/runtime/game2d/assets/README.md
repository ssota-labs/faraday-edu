# game2d assets — put files here

This folder is created by `faraday pack add game2d` as `public/assets/game2d/`.

**Nothing is vendored in the pack** (Kenney packs are large). Download CC0
assets yourself (or let the authoring agent fetch them) and drop them here:

```text
public/assets/game2d/
  sprites/     # characters, items (.png)
  tiles/       # tilesets / tile sheets
  ui/          # buttons, panels
  audio/       # .mp3 / .ogg / .wav
```

## Recommended first downloads (all CC0 — free to use, no attribution required)

| Pack | URL | Use for |
|------|-----|---------|
| Kenney — New Platformer Pack | https://kenney.nl/assets/new-platformer-pack | characters, tiles, items |
| Kenney — UI Pack | https://kenney.nl/assets/ui-pack | buttons, panels |
| Kenney — Digital Audio | https://kenney.nl/assets/digital-audio | SFX / UI blips |
| Kenney — Tiny Dungeon | https://kenney.nl/assets/tiny-dungeon | top-down tiles / RPG-ish |

See `.faraday/packs/game2d/assets.md` for the full catalog and license rules.
