# Orbital Period

Fullscreen 3D STEM lesson scaffolded by the **3d-stem** skill.

## Commands

```bash
pnpm install
pnpm dev          # local preview
pnpm check        # structure gate (from monorepo: node skills/3d-stem/scripts/stem.mjs check)
pnpm build
```

## Authoring

- Edit `src/scene/LessonScene.tsx` — the concept lives here.
- Keep the primary viewport fullscreen 3D; HUD may overlay.
- Do **not** add `@faraday-academy/*` dependencies.
- Optional education UI: copy from the Faraday shadcn registry (`apps/ui`).
