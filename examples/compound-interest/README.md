# Compound Interest — 복리의 폭주

GTM Stage 1 short Wow demo (**S1**): drag principal / rate / years / compounding
and watch compound growth peel away from simple interest.

- Scaffolded with `faraday new compound-interest`.
- Static-deployable (`pnpm build` → `dist/`).
