# Assessment — feed-forward checks, not grades

Assessment here means **diagnostic feedback inside the lesson**, never an LMS
gradebook.

## Match form to the outcome verb

| Outcome verb | Form | In a 3D lesson |
|---|---|---|
| recognize / distinguish | Forced choice with misconception distractors | Overlay prompt + highlight the wrong mental model in-scene |
| compute / estimate | Numeric response with tolerance | Read a value from the sim; type or confirm |
| predict / construct | Commit-then-reveal | Sketch/predict, then reveal the true trajectory/state |
| do / tune / achieve | Performance mission | Hit a goal state inside the scene (visible target) |

## Rules

- Checks must require using the interaction — not the title sentence.
- Hints are feed-forward (“raise eccentricity”, not “wrong”).
- No roster, scores-to-server, or cohort analytics in v1.
- Self-assess overlays are honest: don’t pretend ML grading.

## Anti-patterns

- MCQ-only when the outcome says compute/do/predict.
- Invisible mission goals.
- Closing a lesson with no check at all when a clear outcome was stated.
