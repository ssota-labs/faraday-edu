---
title: Getting started
description: How to use this docs-first workspace for the 3d-stem pivot.
type: guide
---

1. Read the [product vision](/docs/vision) and [GTM](/docs/planning/gtm) —
   Faraday Academy ships the **`3d-stem`** skill.
2. Follow the active [PRD](/docs/planning/prds/prd-3d-stem-interactive-textbook-skill) and
   [pivot plan](/docs/development/plans/plan-pivot-faraday-academy-to-3d-stem).
3. Capture further changes as PRD → story → spec/ADR → plan before code.
4. Implement only against a plan that is already `ready` or `active` on your base branch.
5. Run `pnpm check:planning` / `omd check` before opening an implementation PR.
