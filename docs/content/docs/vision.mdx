## Vision

**Faraday Academy** (repo) ships **`3d-stem`** (skill): coding agents turn STEM
topics into **fullscreen 3D interactive textbooks** so learners can see and
manipulate science, engineering, math, and CS concepts.

Distribution follows img2threejs / Oh My Docs: the skill bundles scripts and
knowledge; authors talk to an agent; there is no public npm CLI product path.
Education UI ships as a **shadcn registry** with an in-repo component library
site.

## Principles

1. **One job** — fullscreen 3D STEM lessons; not an LMS or course marketplace.
2. **Skill-first** — scripts enforce gates; the agent judges clarity and pedagogy.
3. **Pedagogy on demand** — learning-design references load when relevant.
4. **Registry UI, not npm UI** — copy components into the lesson when needed.
5. **Intent precedes implementation** — PRD → story → spec/ADR → plan → code.
